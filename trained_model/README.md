# Trained Model

The trained OpenVLA model produced using the extended LIBERO spatial augmented dataset is hosted externally on Hugging Face because the checkpoint is too large to store directly in GitHub.

Model repository:

https://huggingface.co/MyLastWinterX/openvla-libero-spatial-extended

The uploaded model folder contains:

- model safetensor shards
- tokenizer files
- preprocessor configuration
- dataset statistics
- model configuration
